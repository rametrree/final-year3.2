# 🎓 Final Exam Schedule Dashboard

An interactive web dashboard for managing and visualizing a final exam schedule.

🔗 **Live Demo:**
[https://rametrree.github.io/final-year3.2/](https://rametrree.github.io/final-year3.2/)

---

## 📌 About This Project

This project is a Single Page Application (SPA) designed to help students organize and manage their final exam schedule in an interactive and user-friendly way.

It provides:

* 📊 Exam statistics overview
* 🗓️ Interactive calendar with highlighted exam dates
* 🔍 Search functionality (by subject name or code)
* 🎯 Date filtering
* 🪑 Interactive seat map viewer
* 📌 Subjects without final exams section

## ✨ Features

### 1️⃣ Dashboard Metrics

* Total number of exams
* Total exam days
* Subjects without scheduled final exams

---

### 2️⃣ Interactive Calendar

* Displays exam month
* Highlights days with exams
* Click a date to automatically filter the table

---

### 3️⃣ Search & Filter System

* Search by subject name
* Search by subject code
* Filter exams by selected date

---

### 4️⃣ Seat Map Viewer

* Click "View Seat Map"
* Displays a grid-based classroom layout
* Highlights the user's seat
* Fully responsive design

---

### 5️⃣ Subjects Without Final Exam

Displays subjects evaluated by:

* Projects
* Practical exams
* Alternative assessments


2. Navigate into the folder:

```bash

```

3. Open `index.html` in your browser.
